# GSURL

GSURL is a URL Shortner Tool for Terminals.

This Tool is completely made on Python & It is an Open Source project.

## Installation

Follow below steps to install GSURL.

Open Terminal and paste & run below codes one by one.

Note: Python-3 is needed to install this Tool.

```bash

 git clone https://github.com/goemonmods/gsurl.git

 cd gsurl

 pip  install -r requirements.txt

```

## Usage

To use the tool follow Instruction below.

1. Go the the GSURL folder inside terminal by below code.

```bash

 cd gsurl

```

2. Next run below code.

```python

 python gsurl.py

```

3. Enter your long URL here & press Enter.

![Enter URL](img/enterurl.jpg)

4. Boom !! You have your Shortened URL.
